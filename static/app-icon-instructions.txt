# 应用图标说明

## 应用名称
已修改为：搜题

## 图标要求
- 文字：毛笔书法字体"搜题"
- 风格：中国传统书法风格
- 背景：白色或浅色渐变
- 尺寸：正方形（512x512像素或更高）
- 格式：PNG（透明背景或白色背景）

## 生成方法

### 方法1：使用在线AI图片生成工具
访问以下URL在浏览器中生成：
```
https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=Chinese%20calligraphy%20brush%20style%20characters%20%22搜题%22%2C%20elegant%20traditional%20Chinese%20calligraphy%2C%20on%20white%20background%2C%20square%20format%2C%20high%20quality%2C%20for%20app%20icon&image_size=square_hd
```

### 方法2：使用其他AI绘图工具
- Midjourney
- Stable Diffusion
- DALL-E 3
- 其他国产AI绘图工具

### 提示词（Prompt）
```
毛笔书法字体"搜题"，中国传统书法风格，优雅大气，白色背景，正方形，高质量，应用图标
```

英文版本：
```
Chinese calligraphy brush style characters "搜题", elegant traditional Chinese calligraphy, on white background, square format, high quality, for app icon
```

## 配置图标到项目

生成图标后，按照以下步骤配置：

1. 将生成的图标文件保存为 PNG 格式
2. 将文件命名为：logo.png
3. 替换项目中的 static/logo.png 文件
4. 在 HBuilderX 中，打开 manifest.json
5. 进入 "App图标配置" 选项
6. 上传不同尺寸的图标（Android需要多种尺寸）

## Android图标尺寸要求

- 48x48 (mdpi)
- 72x72 (hdpi)
- 96x96 (xhdpi)
- 144x144 (xxhdpi)
- 192x192 (xxxhdpi)
- 512x512 (Google Play商店)
